## ⬇️How To install?
If you want to install this app and run it locally, please follow these steps:

### 1. Clone the repository
```
git clone https://github.com/Parhamddeveloper/React-Todo-list-App
```
### 2. Navigate to the project directory
```
cd React-Todo-list-App
```
### 3. Install dependencies
```
npm install
```
### 4. Run the App
```
npm run dev
```
you can open the app using this address:

**http://localhost:5173/**

**Note :If you want to use this app as PWA, after installing dependencies, follow these steps :**
### 1. Build the App
```
npm run build
```
### 2. Run the App  
```
npm run preview
```
**Another note: if you want to use this app as PWA, you must open the app using this address:**

**http://localhost:4173**
